Zoid's Announcement Plugin
=============================
Author: Zoidberg
Plugin for TF2CSA, and the Trinity Trade Server, South Africa.
=============================


This plugin:
  Posts an announcement in chat at regular intervals
With:
  Custom Intervals
  Custom Messsages

Fixes:
  0.2 Fixed Exploit with commands being available to admins only now
  0.3 Fixed sm_interval not changing the interval
      Added a README

Aims:
  Multiple Messages to be loaded.

=============================
Available Commands
=============================
sm_message       -Sets the message for the announcement.
sm_interval      -Sets interval between announcementsin seconds.
sm_startmessages -Starts Timer again.
sm_stopmessages  -Stops Timer.


Thank you for downloading.
